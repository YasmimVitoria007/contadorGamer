# 🎮 Contador Gamer

Projeto desonvolvido em React Native com Expo

# Preview do APP

O aplicativo possui
* contador de pontos;
* botão de aumentar;
* botão d ediminuir
* botão de resetar;
* interface estilo gamer

# tecnologias utilizadas

* React Native
* Expo
* JavaScript

# como executar projeto

## 1. clonar o repositorio
``` 
git clone https://github.com/YasmimVitoria007/contadorGamer.git
```

## 2. Entrar na pasta do projeto

```
cd contadorGamer
```

## 3. Instalasr dependências

```
npm install
```

## 4. Rodar Projeto

```
npx start
```
